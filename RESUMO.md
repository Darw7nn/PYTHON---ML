# 🐍 Resumo: Introdução e Conceitos Iniciais de Python

![GIF de um gatinho digitando freneticamente no computador](https://media.giphy.com/media/JIX9t2j0ZTN9S/giphy.gif)

## 1. O que é o Python e sua História 📜
* **O que é:** Python é uma linguagem de alto nível, interpretada, multiplataforma e de propósito geral. Ela é reconhecida pela sua simplicidade, legibilidade e alta produtividade. 🚀
* **História:** Foi criada por Guido van Rossum no final da década de 1980, tendo sua versão 0.9.0 lançada em 1991. 🕰️ O nome da linguagem não vem do réptil, mas sim de uma homenagem ao grupo humorístico britânico *Monty Python*. 🎬
* **Aplicações e Uso:** É amplamente utilizada em Automação, Desenvolvimento Web, Ciência de Dados, Machine Learning e Inteligência Artificial. 🤖 Pode ser executada no modo interativo ou por meio de arquivos de script `.py`. 💻

## 2. Estrutura e Sintaxe Básica 🏗️
* Diferente de outras linguagens, o Python executa suas instruções sequencialmente (de cima para baixo) e não exige a declaração de um método principal como o `main`. ⬇️
* **Comentários:** Códigos podem ser comentados usando `#` para uma única linha ou `"""` (docstrings) para múltiplas linhas. 📝

## 3. Variáveis e Tipos de Dados 📦
* **Variáveis:** Em Python, não é necessário declarar o tipo da variável antes de usá-la, pois o tipo é inferido automaticamente pelo interpretador e pode mudar durante a execução. 🪄
* **Tipos Básicos - Inteiro (`int`):** ex. `10` 🔢
* **Tipos Básicos - Ponto flutuante (`float`):** ex. `19.90` ⏺️
* **Tipos Básicos - Texto (`str`):** ex. `"Python é incrível"` 🔤
* **Tipos Básicos - Booleano (`bool`):** ex. `True` ✅
* É possível usar a função `type()` para descobrir o tipo de um dado armazenado. 🔍

## 4. Entrada de Dados e Conversão (Casting) 📥
* A função `input()` é utilizada para receber dados digitados pelo usuário. ⌨️
* Todo dado recebido através do `input()` é lido como uma string (`str`) por padrão. 🔡
* Para realizar operações matemáticas, é necessário converter essa entrada usando funções de *casting*, como `int()` para inteiros ou `float()` para números decimais. 🔄

## 5. Operadores Aritméticos Básicos 🧮
* A linguagem suporta as operações clássicas: Adição (`+`), Subtração (`-`), Multiplicação (`*`) e Divisão (`/`). ➕
* Também oferece operadores específicos úteis: Divisão inteira (`//`), Resto da divisão (`%`) e Potência (`**`). ⚡

## 6. Boas Práticas Iniciais 🌟
* Escolha nomes de variáveis claros e que descrevam bem o seu propósito. 🎯
* Evite usar acentuação e espaços em branco na nomeação. 🚫
* Utilize o padrão `snake_case`, escrevendo com letras minúsculas separadas por `_` (ex: `media_final`). 🐍
